Contoh Pseudocode: Menemukan Nilai Tertinggi

Pseudocode adalah cara menulis logika program menggunakan bahasa yang mirip bahasa manusia, yang membantu kita merancang alur sebelum menuliskannya dalam kode. Kode ini menunjukkan cara menemukan nilai tertinggi dalam sebuah daftar.

PROGRAM CariNilaiTertinggi
  // Input: sebuah daftar angka
  INPUT: daftarNilai
  // Asumsikan nilai pertama adalah yang tertinggi untuk sementara
  SET nilaiTertinggi = daftarNilai[0]
  
  // Lakukan perulangan untuk setiap nilai dalam daftar
  FOR setiap nilai DALAM daftarNilai:
    
    // Bandingkan nilai saat ini dengan nilai tertinggi yang sudah disimpan
    IF nilai > nilaiTertinggi THEN
      // Jika nilai saat ini lebih tinggi, update nilaiTertinggi
      SET nilaiTertinggi = nilai
    END IF
  END FOR
  
  // Output: tampilkan hasil akhir
  OUTPUT: nilaiTertinggi
END PROGRAM

Peran Manajer Utama: Sistem Operasi

OS bertindak sebagai jembatan antara pengguna dan perangkat keras, mengelola semua sumber daya untuk memastikan efisiensi dan stabilitas. Grafik ini menyoroti berbagai fungsi penting yang dikelola oleh OS.

Glosarium

Kumpulan istilah penting yang digunakan dalam infografis ini.

Algoritma

Serangkaian langkah logis dan terstruktur untuk menyelesaikan suatu masalah.

Struktur Data

Cara menyimpan, mengatur, dan mengelola data di dalam komputer agar dapat digunakan secara efisien.

Pseudocode

Cara menuliskan logika sebuah program menggunakan bahasa yang mirip bahasa manusia, sebagai rancangan sebelum menulis kode.

Sistem Operasi (OS)

Perangkat lunak inti yang mengelola semua sumber daya perangkat keras dan perangkat lunak di komputer.

CPU

Central Processing Unit, otak dari komputer yang mengeksekusi instruksi program.

ALU

Arithmetic Logic Unit. Bagian dari CPU yang melakukan operasi matematika dan logika.

CU

Control Unit. Bagian dari CPU yang mengarahkan aliran data dan instruksi.

RAM

Random Access Memory, memori utama yang berfungsi sebagai penyimpanan data dan instruksi sementara.

Bus

Jalur komunikasi yang menghubungkan semua komponen komputer untuk mentransfer data dan sinyal kontrol.

LIFO

Last-In, First-Out. Prinsip struktur data Stack, di mana data terakhir yang masuk adalah yang pertama keluar.

FIFO

First-In, First-Out. Prinsip struktur data Queue, di mana data pertama yang masuk adalah yang pertama keluar.
